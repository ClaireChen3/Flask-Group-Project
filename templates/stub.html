Project Plan - Wordle Game

- Product similar to online game "Wordle"

- Normal mode:

unknown five letter word, six guesses, each guess: letters that are in the word and in the correct placement appear green, letters that are in the word but not in the correct space appear blue, letters that are not in the word at all appear grey

- Change the mode:

change the number of letters in the words, change number of guesses allowed, specific genre of words, set time limit

- Things to implement:

clear words/start over, hint(reveal letter, etc), statistics after each game

- Know how to implement:

working keybaord, designed webpage, user input for submitted letter, animations after submitting letter
